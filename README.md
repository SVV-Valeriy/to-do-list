# To Do List

> Проект предстовляет собой приложения, которое позволяет пользователю добавлять, удалять, сортировать и искать задачи, что делает его удобным списком дел.

## Built With:
 - HTML
 - CSS
 - TypeScript
 - React
 - Redux ToolKit
 - Formik
